# SU Flight Simulator
## CSC361 Project 3 Group 6

## Controls
<kbd>Q</kbd>/<kbd>E</kbd> Throttle up/down

<kbd>S</kbd>/<kbd>W</kbd> Pitch up/down

<kbd>A</kbd>/<kbd>D</kbd> Bank left/right
